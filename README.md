# NexusChat

Мессенджер для небольшого сообщества с личными сообщениями, групповыми чатами и каналами.

## 🚀 Быстрый старт

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск сервера
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

## 📋 API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/` | Страница входа/регистрации |
| GET | `/chat` | Основной интерфейс чата |
| POST | `/api/register` | Регистрация пользователя |
| POST | `/api/login` | Вход (JWT токен) |
| GET | `/api/me` | Профиль текущего пользователя |
| PATCH | `/api/me/profile` | Обновление профиля |
| POST | `/api/upload` | Загрузка файла |
| POST | `/api/me/avatar` | Загрузка аватара |
| GET | `/api/pinned` | Закреплённые сообщения |
| WS | `/ws` | WebSocket для чата |

## 🔧 Технологии

- **Backend**: FastAPI + SQLAlchemy
- **Database**: SQLite
- **Auth**: JWT (python-jose) + bcrypt
- **Frontend**: Vanilla JS + CSS Variables
- **Real-time**: WebSocket

## 📁 Структура проекта

```
nexuschat/
├── main.py              # FastAPI приложение
├── models.py            # SQLAlchemy модели

│   └── chat.html        # Интерфейс чата
├── uploads/             # Загруженные файлы

```

## 🔐 Безопасность

- Пароли хешируются через bcrypt (12 раундов)
- JWT токены с истечением срока
- Валидация типов загружаемых файлов
- Защита от XSS (escape HTML на клиенте)

## ⚠️ Для production

1. Замените `SECRET_KEY` в `auth.py` на случайную строку
2. Используйте PostgreSQL вместо SQLite
3. Настройте HTTPS
4. Добавьте rate limiting
5. Включите логирование
